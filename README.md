SCNUOJ 派生自 JNOJ，是一个在线判题系统。

SCNUOJ 可对用户在线提交的源代码进行编译和执行，并通过预先设计的测试数据检验代码的正确性。

目前 SCNUOJ 主要用于华南师大软件学院程序设计类课程教学、ACM 训练和比赛等，由软件协会维护。

请访问 [项目维基](https://github.com/SCNU-SoCoding/scnuoj/wiki) 了解更多信息，包括功能特性、部署说明等。
